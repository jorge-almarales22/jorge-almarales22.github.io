EZ-WIN

Ez-win es una empresa que está asociado a los concesionarios de USA y esta le ayuda a hacer un seguimiento de sus ventas de autos y repuestos.

Aloa

  • Industria: Tecnologías
  • Equipo: 4
  • Sitio web: ez-win.
El principal requisito de EZ-WIN era poder llevar seguimiento a todas las actividades dentro de los concesionarios que estaban afiliados a ellos, llevar la contabilidad de las ventas de autos y repuestos, hacer seguimiento a las garantías de los vehículos y partes vendidas. Todo esto en tiempo real y sin perdidas de datos subiendo los archivos vía API o por archivos CSV.

Requisitos del proyecto

  • Que sea accesible.
  • Portabilidad.
  • Confibilidad en los datos.

Solución

La solución se implementó de la siguiente manera. Se utilizó React para crear una interfaz de usuario moderna y responsiva. Para el backend Se utilizó Node para crear un servidor web que sirviera la aplicación React. El servidor se implementó utilizando Express, un framework web popular para Node. En base de datos Se utilizó MariaDB para almacenar los datos de las ventas, el inventario y los análisis. La base de datos fué MariaDB y la app se configuró utilizando Docker

Los resultados

Ez Win es una solución exitosa para la gestión de ventas de vehículos. La aplicación es fácil de usar, escalable y segura. La aplicación ha tenido un impacto positivo en los concesionarios afiliados a la empresa.
